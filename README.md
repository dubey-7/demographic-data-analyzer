# Demographic Data Analyzer

This project analyzes demographic data using Pandas.  
The dataset used in this project was extracted from the 1994 U.S. Census database.

The Demographic Data Analyzer is an interactive Python-based data analysis project that explores real-world census data to uncover meaningful demographic insights. Instead of just viewing raw data, this project processes and analyzes it to answer important analytical questions about income, education, race distribution, working hours, and country-wise earnings.
